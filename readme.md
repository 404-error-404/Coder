## 一些算法总结
- [递归加回溯](./src/leetcode/So05/So0508.java)
  
  使用递归在每次操作后调用自身，以此记录每次的状态，然后在递归中判断当前状态是否可行，可行就直接返回，不可行返回递归上一层继续执行
